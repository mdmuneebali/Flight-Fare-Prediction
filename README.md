# Flight Fare Prediction

This project predicts flight ticket prices using Machine Learning based on key flight features.

## Project Overview
The goal is to build a regression model that estimates airfare prices using historical flight data.

## Key Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training & Evaluation

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## How to Run
1. Open the notebook: `flight_fairs.ipynb`
2. Run all cells in Jupyter Notebook / Google Colab
